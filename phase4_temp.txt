PHASE 4B — HEALTH CHECK POLICY
Check types:
1. PROCESS CHECK (alive?)
2. PORT CHECK (reachable?)
3. ROUTE CHECK (expected status?)
4. FUNCTIONAL CHECK (does the expected job?)
5. DEPENDENCY CHECK (downstream available?)
A service is healthy only if heartbeat + functional check both pass.

PHASE 4E — CI TEST ENFORCEMENT

PHASE 4F — INCIDENT CLASSIFICATION
